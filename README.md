**一些说明** 
 
 

> 1. 双击 当前分类名: 可以回到当前分类顶部。
> 2. 长按 分类名收藏: 清空所有收藏。（v21已改版）
> 3. 部分分类支持长按分类名立即更新当前分类数据。（默认一个小时更新一次）*v21版本后 除篮球足球 6分钟 其它默认半小时*
> 4. 若视频卡顿、黑屏，可尝试换线路或者刷新。
> 5. 在视频控制条里，如果有 ☆ 五角星图标， 点击即可收藏！有部分频道是不支持收藏的。
